核心功能

从记录到整理，再到反馈，形成一条完整的自我管理闭环。

Dual Capture

双通道记录：想法与行动分开管理

把“脑内反复想的事”和“现实要处理的事”分别记录，不再混在一起。信息更清晰，执行更有方向。

State Flow

待处理 / 已完成，一键切换

每条记录都能快速切换状态，今天该做什么、已经推进了什么，一眼就能看懂，减少拖延和内耗。

Visual Journal

视觉留存：把瞬间变成可回看的素材

拍下或上传关键画面，自动沉淀为视觉片段。回看时更容易找回当时的场景、情绪与灵感。

Audio Space

声音空间：在播放中放慢思绪

用声音建立一个可随时进入的专注区。无论是放松、复盘还是重启状态，都有一个稳定的“过渡带”。

Progress Insight

多维统计反馈：看见你正在变好

系统会汇总不同维度的记录与完成进度，用直观数据帮助你识别节奏、调整策略，让成长不再只靠感觉。
