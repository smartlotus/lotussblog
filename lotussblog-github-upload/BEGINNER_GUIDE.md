# 傻瓜式网页维护指南（零基础）

## 你只要记住两个文件
- `manage-site.bat`：双击后进入管理菜单，用来新增/删除/编辑内容。
- `publish-site.bat`：双击后一键同步内容（不进菜单，直接发布到本地网页文件）。

## 第一次使用（一步一步）
1. 打开项目文件夹：`C:\Users\28389\Desktop\elliptical-rotation-chart-master`
2. 双击 `manage-site.bat`
3. 看到菜单后，输入数字并按回车：
   - `1` 一键发布
   - `2` 新增内容
   - `3` 删除内容
   - `4` 编辑内容（文字/图片/排版）
   - `5` 编辑首页标题
   - `6` 查看当前内容列表
   - `7` 导入本地 HTML（把你自己写好的网页文件直接加进来）
   - `8` 添加外部网页链接（直接跳转到网站）
   - `0` 退出

## 如何新增一页内容（最常用）
1. 双击 `manage-site.bat`
2. 输入 `2`，进入“新增内容向导”
3. 按提示依次填写：
   - 卡片标题
   - 页面 slug（可回车自动生成）
   - 页面主标题
   - 页面简介
   - 正文（多段用 `|` 分隔）
   - 排版（1/2/3）
   - 卡片图片路径（可拖拽图片到窗口）
   - 页面图片路径（可回车默认同卡片）
4. 完成后脚本会自动生成：
   - 轮播卡片配置：`config.js`
   - 页面文件：`pages/content/<slug>.html`
5. 刷新 `index.html` 即可看到新增内容

## 如何直接导入你本地写好的 HTML（新增功能）
1. 双击 `manage-site.bat`
2. 输入 `7`
3. 按提示填写：
   - 本地 HTML 路径（可直接拖文件）
   - slug（可回车自动）
   - 卡片标题
   - 卡片图片
4. 导入后会自动：
   - 把 HTML 复制到 `pages/content/<slug>.html`
   - 加入轮播卡片
   - 立即同步到 `config.js`

## 如何添加外部网页链接（新增功能）
1. 双击 `manage-site.bat`
2. 输入 `8`
3. 按提示填写：
   - 外部网页链接（`https://...`）
   - 卡片标题
   - 卡片图片
4. 完成后轮播会新增一个卡片，点击后直接跳转外部链接

## 如何删除内容（一键删减）
1. 双击 `manage-site.bat`
2. 输入 `3`
3. 选择要删除的序号
4. 输入 `DELETE` 确认
5. 根据提示选择是否删除关联图片

## 如何改文字、改图片、改排版
1. 双击 `manage-site.bat`
2. 输入 `4`（编辑现有内容）
3. 选择内容序号
4. 按提示逐项修改：
   - 文字：标题、简介、正文
   - 图片：卡片图、页面图
   - 排版：图片左/图片右/纯文字
5. 回车可保持原值，不会覆盖

## 日常维护建议（长期稳定）
1. 所有“正式内容页”都放在 `pages/content`
2. 轮播图片放在 `asset/image/cards`
3. 页面图片放在 `asset/image/pages`
4. 每次改完都执行一次：
   - 菜单中输入 `1`，或者双击 `publish-site.bat`
5. 脚本会自动备份到 `backups`，出错可回退

## 常见问题
- 图片不显示：确认路径正确，或直接拖图片文件到输入框。
- 菜单中文乱码：直接双击 `manage-site.bat`（已自动切换 UTF-8）。
- 想恢复旧版本：去 `backups` 文件夹找对应时间的备份文件。
