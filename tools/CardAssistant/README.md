# Card Assistant (New)

This is a brand-new standalone visual card manager.

## Launch

Run:

`run-card-assistant-new.bat`

## What it does

- Delete card and sync immediately to website files.
- Add card via external URL.
- Add card via local `.html/.htm` file import.
- Customize card title and card image.
- Edit existing card settings and sync immediately.
- Save backups under `backups/card-assistant-new`.
